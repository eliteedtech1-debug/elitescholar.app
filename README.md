# Elite Scholar Android App

Official Android application for Elite Scholar LMS.

## 📥 Download

**Latest Version:** v1.0.0  
**[Download APK](https://github.com/eliteedtech1-debug/elitescholar.app/releases/latest/download/EliteScholar.apk)**

## 📱 Installation

1. Download the APK from the link above
2. Enable "Install from unknown sources" in your device settings
3. Open the downloaded APK file
4. Tap "Install"

## ✨ Features

- Complete LMS functionality
- Student, teacher, and parent dashboards
- Payment and billing system
- Academic management
- Attendance tracking
- Exam and results management
- Fingerprint authentication
- Auto-update notifications

## 🔄 Auto-Update

The app automatically checks for updates on startup. When a new version is available, you'll be prompted to download and install it.

## 📋 System Requirements

- Android 6.0 (API 23) or higher
- Internet connection required for data sync
- Fingerprint sensor (optional, for biometric login)

## 📞 Support

For issues or questions, contact: support@elitescholar.ng

## 🔒 Privacy

This app requires internet connection to sync data with your school's server. All data is encrypted and secure.
